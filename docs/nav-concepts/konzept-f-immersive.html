Konzept F

Immersive Full-Screen Navigation

Die Navigation öffnet sich als Fullscreen-Overlay über die komplette Seite. Links riesige, edle Typografie für die Hauptpunkte mit Index-Nummern. Rechts eine Preview-Karte, die sich beim Hover über die Links ändert. Jede Karte hat einen generativen abstrakten Hintergrund, der beim Hover Farbe annimmt (Grayscale → Color). Unter den Hauptlinks: Sublinks als Grid. Das gesamte Interface nutzt mix-blend-mode: difference für den Header, damit er auf allen Hintergründen sichtbar bleibt.
