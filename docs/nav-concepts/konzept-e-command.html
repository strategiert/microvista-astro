⌘ Konzept E — Command Palette

Navigation per Tastatur

Drücken Sie ⌘K (oder klicken Sie die Suchleiste) — ein Spotlight-ähnliches Interface öffnet sich. Suchen, filtern, navigieren. Alles ohne Maus.

E

Konzept E — So funktioniert es

Die Navigation wird durch eine persistente Command Bar im Header ausgelöst. ⌘K (oder Klick) öffnet die Command Palette — ein modales Overlay mit Suchfeld, Filter-Pills für Bereiche (Labor/Serie/Branchen/Wissen), und einer scrollbaren Liste aller Seiten mit Tastatur-Shortcuts. Jeder Eintrag hat Icon, Titel, Beschreibung und Bereich-Badge. Die Palette ist vollständig Tastatur-navigierbar (Pfeiltasten, Enter, ESC). Unter den Ergebnissen: Quick Actions für häufige Aufgaben.
